    Help

    How to use the search bar?

    All you have to do is enter a value for any possible column of a given table entry into the search bar, and the table will only display the table entries that have a value matching what you entered into the search bar.

    How to add new item?

    To add new item simply click on the 'Add New Item' button.

    That will bring up a pop up form which you can fill out and add the item to the table by clicking on the 'Add item' button. All attributes must be filled, no value can be null.

    How to edit an item?

    To add new item simply click on the 'Edit Item' button. That will bring up a pop up form that is very much like the 'Add New Item' form. The difference being that if requires a barcode value that already exists on the table and all the other attribute values entered replace the attribute values of the item with corresponding barcode. Click the 'Edit Item' button to submit your chanes. All attributes must be filled, no value can be null.

    How to log in?

    When you open the website you will arrive at the log in page. Simply type in your Username and Password, just like you would on any other website, and click the 'Log In' Button.
